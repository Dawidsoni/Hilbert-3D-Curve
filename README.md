# Hilbert3D curve

## Generating curve:

`python hilbert3d.py curve_order image_size curve_scale camera_dist curve_pos_x curve_pos_y curve_pos_z ox_angle oy_angle > output.ps`

## Sample usage:

`python hilbert3d.py 4 500 20 600 400 400 400 10 40 > example1.ps`
